# NexusOne Bot Framework

Framework xây dựng chatbot đa nền tảng

## Các nền tảng hỗ trợ
- Facebook Messenger
- Zalo OA
- Telegram
- TikTok
- Weibo (coming soon)

## Cấu trúc dự án
\`\`\`
$(tree -L 3 NexusOne)
\`\`\`

## Cài đặt
1. Sao chép file .env.example thành .env
2. Cập nhật thông tin cấu hình
3. Cài đặt dependencies: \`npm install\`
4. Khởi chạy: \`npm start\`
